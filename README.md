## Why was Slide Animation made

This Slide Animation was made to help web developers create an easy slide animation in - ease. Simply with Html and Css.

## Running the project

1. Clone the repo.
2. Run `npm install`.
3. Run `npm start`.
4. Navigate to `http://localhost:3000`.
